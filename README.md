# 🤖 Bare-Bones Agentic AI with Gemini 3.1

A minimalist, high-performance implementation of an **AI Agent** using the `gemini-3.1-flash-lite-preview` model. This repository demonstrates the core "Reasoning Loop" (ReAct) where the AI autonomously decides to use Python functions to solve tasks.

## 🚀 Why this Repo?
Most Agent frameworks (like LangChain) add a lot of complexity. This project uses the **native Google GenAI SDK** to show you exactly how an agent works under the hood with zero "magic" libraries.

* **Model:** Gemini 3.1 Flash-Lite (Optimized for 2026 speed/cost)
* **Pattern:** Automated Function Calling (Agentic Loop)
* **Cost:** $0 (Uses Google AI Studio Free Tier)
